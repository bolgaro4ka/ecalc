## Требования
- Компьютер (Электронно-вычислительная машина, умеющяя запускать Linux/Windows 8+)
- <a href="https://nodejs.org/" >Node JS</a> 
- Electron (`npm i` или `npm i electron`)

## Запуск 
`npm run dev`

## Сборка
`npm run start`
`npm run make`
